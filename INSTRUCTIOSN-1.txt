Build a complete Python script: `trending_india_pipeline.py`

## PURPOSE
Fetch trending topics from India, expand them into product-intent signals 
for a dropshipping pipeline, and export structured JSON.

## EXACT OUTPUT FORMAT
File: `trending_india_<YYYY-MM-DD>.json`
Schema:
{
  "metadata": {
    "scraped_at": "<ISO timestamp>",
    "geo": "IN",
    "total_topics": <int>,
    "sources": ["google_trends_realtime", "google_trends_daily", "youtube_trending"]
  },
  "trends": [
    {
      "rank": 1,
      "topic": "<string>",
      "traffic_estimate": "<string>",  // e.g. "500K+" if available, else "N/A"
      "source": "<source_name>",
      "related_queries": ["<string>", ...],  // min 3, max 10
      "product_intent_score": <0.0-1.0>,     // heuristic, not AI
      "suggested_categories": ["<string>"]   // derived from keywords
    }
  ]
}

## SOURCES TO IMPLEMENT (in priority order)

### 1. pytrends-modern (PRIMARY)
- Install: pip install pytrends-modern
- realtime_trending_searches(pn='IN') → parse all story titles + articles
- trending_searches(pn='india') → daily hot searches
- For each trend topic, call related_queries() with geo='IN', timeframe='now 1-d'
- Add 5s sleep between related_queries() calls to avoid rate limiting
- Wrap every pytrends call in try/except — if rate limited, skip and continue

### 2. YouTube Data API v3 (SECONDARY)
- Endpoint: https://www.googleapis.com/youtube/v3/videos
- Params: chart=mostPopular, regionCode=IN, maxResults=50, part=snippet
- API key read from env var: YOUTUBE_API_KEY
- Extract: video title, tags, category
- If YOUTUBE_API_KEY not set → skip silently, log warning

### 3. Google News RSS (TERTIARY)  
- URL: https://news.google.com/rss?hl=en-IN&gl=IN&ceid=IN:en
- Parse with feedparser
- Extract top 50 entry titles
- Use as additional topic signals

## PRODUCT INTENT SCORING (no AI, pure heuristic)
Score 0.0–1.0 based on keyword presence:

High intent keywords (score += 0.3 each, max 1.0):
  buy, price, cheap, best, review, order, shop, deal, discount,
  under 500, under 1000, combo, kit, pack, wireless, portable,
  india 2026, trending, new, latest, vs, comparison
  - Immediate Transactional (Highest Intent: Wallet is out)
  sale, offer, coupon, promo code, clearance
  cash on delivery, cod, free shipping, emi
  in stock, where to buy, lowest price, cheap price
  wholesale, bulk, dropship, distributor
  near me (when combined with a product name)
  - Commercial Investigation (High Intent: Narrowing down options)
  alternative to, similar to, replacement for
  specs, specifications, features, dimensions
  unboxing, hands on, teardown, test
  pros and cons, is it worth it, buying guide
  top 10, top 5, tier list, ranking
  benchmarks (especially useful for hardware/electronics)
  reddit, quora (users often append these to bypass SEO spam and find real reviews)
  - Product Status & Trust Verifiers (Medium-High Intent)
  original, authentic, genuine, fake vs real
  refurbished, second hand, used, open box
  warranty, return policy, customer service
  size chart, colors, variants, edition
  - Urgency & Trend Indicators (High Intent: Time-sensitive)
  flash sale, limited time, lightning deal
  restock, sold out, in stock tracker
  pre-order, launch date, release date, early access
  -Negative Intent (Subtract Score: Informational/DIY)
  how to, tutorial, diy, repair, fix, not working, error, broken, manual, pdf, driver download, free download

## DEDUPLICATION
- Normalize: lowercase, strip punctuation, remove stopwords (a, the, in, of, for, with, is, are, was)
- If two topics have >70% word overlap → keep the one with more related_queries
- Final output: deduplicated, sorted by product_intent_score DESC

## ERROR HANDLING
- Every network call wrapped in try/except with specific error messages
- If pytrends completely fails → still run YouTube + RSS sources
- Log all errors to: `pipeline_errors_<date>.log`
- Script should NEVER crash — always produce output even if partial
